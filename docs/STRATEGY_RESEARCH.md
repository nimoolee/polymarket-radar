# 策略研究记录

## 当前策略假设

1. 高概率本身没有意义，必须结合 CLOB 可交易价格、spread、流动性和剩余时间。
2. Gamma 与 CLOB 差异大时，应标记为数据滞后风险，而不是直接视为交易机会。
3. 体育/电竞开赛后仍可能存在机会，但必须持续校验盘口和真实比赛状态。
4. 天气、政治、外交、By Date 市场需要不同时间解释逻辑。

## 当前评分逻辑

评分综合：

1. CLOB 概率。
2. 剩余时间或资金占用时间。
3. 流动性。
4. spread。
5. 30s / 3m 价格动量。
6. 盘口新鲜度和结束风险。

## 风险

1. Gamma 价格滞后。
2. CLOB 盘口旧或无真实可交易 ask/bid。
3. 体育比赛真实状态缺失。
4. 天气和 By Date 市场的结算规则复杂。
5. 单 CLOB 连接 token 上限会导致低优先级市场更新不及时。

## 当前结论

当前版本只能作为人工观察和机会筛选工具，不能作为自动交易系统。下一步必须接入外部真实数据源和本地持久化，才能做更可靠的 edge 判断和回测。

## 被否定的假设

1. 只看 Gamma outcomePrices 判断机会。
2. 只按概率从高到低排序。
3. 用统一 endDate 解释所有市场。
4. 用一个 WebSocket 订阅所有 token。
